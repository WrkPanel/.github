# Welkom bij WrkPanel 🚀

WrkPanel is het centrale zenuwstelsel voor moderne organisaties. Wij geloven in software die meewerkt, niet tegenwerkt.

### Wat wij doen:
- **CRM & Relatiebeheer:** Al je contacten en interacties op één plek.
- **Projectmanagement:** Agile werken met een focus op resultaat.
- **Financiële Ecosystemen:** 
    - **Boekhouding:** Directe koppeling met 200+ pakketten (zoals Exact, AFAS, Visma) via **Combidesk**.
    - **Banken:** Real-time transacties en saldo-inzicht via de beveiligde API van **Ponto**.
    - **Betalingen:** Naadloze factuurverwerking via **Mollie**.
